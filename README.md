# 🤖 HR Chatbot using LLM (Hugging Face)

This project is a simple HR assistant chatbot built using the `google/flan-t5-base` model from Hugging Face, designed to answer questions based on company policy.

## 📚 Features
- Chatbot that responds to questions about HR policies (leave, maternity, working hours).
- Maintains a conversation flow with user name, greetings, and closing.
- Built in Google Colab using `transformers` library.
